# Engage Project

A video and chat conferencing app implementated using WebRTC, Socket.io and Node.js for the Microsoft Engage Mentorship Project. The project has been deployed on herokuapp.

# Features

- Video On/Off
- Audio On/Off
- Screen sharing
- Chat Space
- Leave Room
- Multiple Participants
- Same Server
